grid-area로 이름을 짓고,
grid-template-areas으로 레이아웃을 만든다.

Grid 1

grid-column-start, grid-column-end,
grid-row-start, grid-row-end와
숫자를 이용해서 레이아웃을 만든다.

Grid 2

grid-column-start: 1;
grid-column-end: -1;
grid-column: 1 / -1; 로 축약해서 쓸 수 있다.
-row도 마찬가지이다.

columns와 rows 속성에 대괄호를 사용해서 [naming] 이름 지을 수 있다.
grid-template-columns: [first-line] 100px;
grid-column: first-line / -1;

Grid 3

모바일이나 반응형 디자인이 많기 때문에 px 보다는 fr(fraction, 분수) 단위를 사용한다.
grid-template에서 repeat은 사용할 수 없다. (많이 사용하는 속성인지는 잘 모르겠다.)

Grid 4

flex에서 사용하는 justify-content, align-content 속성이 grid에도 있다.
place-content: align-content / justify-content;
place-items: align-items / justify-items;
place 속성으로 축약이 가능하다.

-content는 grid가 적용된 컨테이너 위치를 조절하고,
-items는 자식 요소들의 셀 안의 위치를 조절한다.
